## Use cases

Remote supports these use cases:   

- Creating a directory structure on remote servers and transferring files to and servers.
- Installing software on remote servers using a package manager.  
- Control programs running on remote servers.  

Remote also presents a Restful API.  The Swagger interface can be used for performing ad-hoc commands on a remote server.
